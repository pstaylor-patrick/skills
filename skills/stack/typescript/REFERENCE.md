# TypeScript Reference

## tsc commands

- `tsc --noEmit` -- type-check without emitting
- `tsc --noEmit --watch` -- watch mode type-check

## Common strict mode errors

- `TS2339` -- property does not exist: add to type or narrow
- `TS2345` -- argument not assignable: check nullability
- `TS2532` -- possibly undefined (noUncheckedIndexedAccess): guard the index access
